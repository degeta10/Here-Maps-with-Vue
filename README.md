# Here Maps with Vue

## Project setup
```
npm install
copy .env.example to .env
replace VUE_APP_HERE_MAPS_API_KEY with your api key
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
